# Isiah Udofia - Resume

This repository contains multiple versions of my resume in different formats.

## Files

### Resume Versions (Non-LaTeX)
- `resume_original.pdf` - The original PDF resume that was converted to LaTeX
- `resume_latest.pdf` - Most recent PDF version
- `resume_consulting.docx` - Word document version tailored for consulting applications

### LaTeX Version
- `isiah_udofia_resume.tex` - LaTeX source file with improved formatting and spacing control

### Documentation
- `README.md` - This file

## LaTeX Features

The LaTeX version includes:
- Consistent, professional spacing and typography
- Clickable hyperlinks (email, LinkedIn, website, portfolio links)
- Easy customization of margins, colors, and formatting
- Better font consistency across different systems

## Compiling the LaTeX

To compile the LaTeX resume:

### Using Overleaf (Recommended)
1. Go to [overleaf.com](https://overleaf.com)
2. Upload the `isiah_udofia_resume.tex` file
3. It will compile automatically with pdfLaTeX

### Local Compilation
```bash
pdflatex isiah_udofia_resume.tex
```

The LaTeX version is optimized for pdfLaTeX compilation and should work seamlessly in Overleaf.
