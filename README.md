## Compiling the Resume

### Using Overleaf (Recommended)
1. Go to [overleaf.com](https://overleaf.com)
2. Upload `isiah_udofia_resume.tex`
3. Compiles automatically with pdfLaTeX

### Local Compilation
```bash
pdflatex isiah_udofia_resume.tex
```
